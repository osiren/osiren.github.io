PC 通过外置网卡分享网络热点

2020-06-30
note
约 295 字 预计阅读 1 分钟 次阅读

文章目录

    • 一、以管理员模式运行 cmd
    • 二、设置 Internet 连接共享
    • 三、注意

办公室所在楼层，仅设置一个路由器，我的办公室手机连接无线网，网速较差，于是想用电脑分享热点。

一、以管理员模式运行 cmd

1
2
3

netsh wlan show drivers  #检查无线网卡是否已插上  
netsh wlan set hostednetwork mode=allow ssid=WIFI热点名 key=WIFI热点密码              #设置WiFi热点名称以及密码
netsh wlan start hostednetwork    #开启热点

二、设置 Internet 连接共享

打开“控制面板”的“网络和共享中心”窗口，单击已连接到Internet的网络连接，选择“属性”→“共享”，勾上“允许其他······连接(N)”并选择“本地连接*9”(这个名字在你执行上述命令后会出现在网络连接面板中)。

三、注意

注意：这些操作，重启后就会失效，不过如果想要重启后再打开的话，在 cmd 中输入 netsh wlan start hostednetwork 即可。
